引脚说明：
PC0-7：数码管a-g和dotpoint
PF6-8：138译码器，对应P22-24，译码器1-3位
PG0-3：按键。key1:加，key2:减，key3:模式切换（闹钟设置模式，设置时间模式），key4:确定键

函数说明：
1. 计时函数。获取RTC的数据然后转换成三个数字，可以让ledTimeShow获取
2. 显示函数。2ms扫描一次数码管，并输出相应数字，同时检测按键。
3. 按键函数

0->a, b, c, d, e, f
1->b, c
2->a, b, g, e, d
3->a, b, c, d, g
4->b, c, f, g
5->a, f, g, c, d
6->a, f, g, c, d, e
7->a, b, c
8->a, b, c, d, e, f, g
9->f, a, b, g, c

tips:
1. keil使用的编译器不支持0b这样的二进制字面值写法
